# vibe

Landing page for the **vibe** project suite — GPU-accelerated geospatial libraries for Python.

## Projects

| Project | Description |
|---------|-------------|
| [vibeSpatial](https://jarmak-personal.github.io/vibeSpatial/) | GPU-accelerated spatial analytics |
| [vibeProj](https://jarmak-personal.github.io/vibeProj/) | GPU-accelerated coordinate projection |
| [vibeSpatial-Raster](https://jarmak-personal.github.io/vibeSpatial-Raster/) | GPU-accelerated raster processing |

## Local Development

This site is built with [Sphinx](https://www.sphinx-doc.org/) and the [Furo](https://pradyunsg.me/furo/) theme.

```bash
uv sync
uv run sphinx-build docs docs/_build/html
```
